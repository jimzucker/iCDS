iCDS — Calc tab color exploration (v2 with reordered output rows)

Same Calc-tab content, four color variants. Trade is NA / 100 bp coupon / 5Y / $10M / R=40% with Quoted Spread = 1000 bp (distressed). Scroll horizontally to compare all four.

Layout change vs current app: Trade Date / Settle Date moved DOWN (rarely changed — defaults are today / T+1). Accrued + Upfront Fee moved UP, right under the headline cards, so the cash split is read alongside Dirty Upfront. Start + Maturity sit between accrued/fee and the chart (they bound the period the upfront covers). New top-to-bottom: Inputs → Quoted/Dirty cards → Accrued + Upfront Fee → Start + Maturity → Default-Risk chart → CS01/DV01/Rec01 → Trade + Settle dates

Current — everything orange

Brand color does five jobs at once: tab icon, chart bar, header label, input value, every computed result. Visual fatigue.
